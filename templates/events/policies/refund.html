1. Event Cancellation by Organizer

If an event is cancelled by the organizer, a full refund will be initiated to the original payment method within 5-7 working days through Razorpay.

2. User Cancellations

Refund policies for attendee cancellations vary by event. Please check the specific event details before registering. If a refund is eligible, it will be processed within the standard timeframe.

3. Refund Process

All refunds are processed back to the original source of payment used during the transaction on Razorpay. You will receive an email notification once the refund is initiated.

4. No-Shows

Failure to attend an event without prior cancellation (as per the event's policy) does not qualify for a refund.
